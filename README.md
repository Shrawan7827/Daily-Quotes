# ✨ Daily Quotes

A simple and interactive **Random Quote Generator** built using **HTML, CSS, and JavaScript**.
The application fetches random quotes from an API and displays them on the screen with the author's name.

Users can generate new quotes, copy them, listen to them using speech synthesis, and share them on social media.

---

## 🚀 Features

* 🎲 Generate a random quote with a button click
* 🧠 Fetch quotes from an external API
* 🔊 Text-to-Speech feature to hear the quote
* 📋 Copy quote to clipboard
* 📱 Share quote via social media
* ⚡ Fast and lightweight web application

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Functionality and API calls
* **Quotable API** – Source of random quotes
* **Web Speech API** – Text-to-speech feature

